# Flutter REX — Design System

## Couleurs (RexColors — theme.dart)

- Accent: #E5484D
- Background dark: #0A0A0A
- Surface: #141414
- Border: #2A2A2A
- Text primary: #FAFAFA
- Text muted: #888888

## Typography

- Font: System default (SF Pro sur macOS)
- Titles: 18px, w600
- Body: 14px, w400
- Captions: 12px, w400, muted

## Spacing (4px grid)

- XS: 4px, S: 8px, M: 16px, L: 24px, XL: 32px

## Règles UI

- Jamais de couleurs hardcodées — toujours via RexColors
- Sidebar items: hauteur 44px, padding H 16px
- Cards: border-radius 8px, border 1px #2A2A2A
- Boutons primaires: accent #E5484D, radius 6px
- Pas de Material elevation/shadows — flat UI uniquement
